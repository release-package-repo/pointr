## 安装

### Windows

1. 下载 `Pointr-0.0.1-windows.exe`
2. 双击运行安装向导
3. 安装完成后从桌面或开始菜单启动 Pointr

### macOS

1. 下载 `Pointr-0.0.1-macos.dmg`
2. 打开 DMG，将 Pointr 拖拽到 Applications 文件夹
3. 首次打开需要在系统偏好设置中允许运行

## 使用

1. 将 USB 接收器插入电脑
2. Pointr 自动识别设备并显示连接状态
3. 通过侧边栏导航切换设置页面
4. 支持中英文界面，可在设备管理页面切换语言

## 系统要求

- **Windows**: Windows 10 2004+ (64-bit)
- **macOS**: macOS 12+
